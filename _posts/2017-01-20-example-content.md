## Inline HTML elements | 行内 HTML 元素

HTML defines a long list of available inline tags, a complete list of which can be found on the [Mozilla Developer Network](https://developer.mozilla.org/en-US/docs/Web/HTML/Element).

HTML 定义了可用的行内标签的长列表，完整列表可以在[Mozilla 开发者网络](https://developer.mozilla.org/en-US/docs/Web/HTML/Element)上找到。

- **To bold text**, use `**To bold text**`. | **加粗文本**，使用 `**加粗文本**`。
- _To italicize text_, use `*To italicize text*`. | _斜体文本_，使用 `*斜体文本*`。
- Abbreviations, like HTML should be defined like this `*[HTML]: HyperText Markup Language`. | 缩写，例如 HTML 应该这样定义 `*[HTML]: HyperText Markup Language`。
- Citations, like <cite>&mdash; Mark otto</cite>, should use `<cite>`. | 引用，例如<cite>&mdash; Mark otto</cite>，应该使用 `<cite>`。
- ~~Deleted~~ text should use `~~deleted~~` and <ins>inserted</ins> text should use `<ins>`. | ~~删除~~文本应使用 `~~删除~~` 和<ins>插入</ins>文本应使用 `<ins>`。
- Superscript <sup>text</sup> uses `<sup>` and subscript <sub>text</sub> uses `<sub>`[^1]. | 上标<sup>文本</sup>使用 `<sup>` 和下标<sub>文本</sub>使用 `<sub>`[^1]。

Most of these elements are styled by browsers with few modifications on our part. | 这些元素中的大多数由浏览器样式化，我们只做了少量修改。
